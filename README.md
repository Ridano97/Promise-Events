# Promise Events

Projet Next.js vitrine premium pour Promise Events.

## Lancer le site

```bash
npm run dev
```

Le site est disponible sur `http://localhost:3000`.

## Structure

- `src/app/page.jsx` assemble la page.
- `src/components/*.jsx` contient les sections.
- `src/components/*.css` contient les styles par section.
- `public/images` contient les vrais visuels renommés et prêts pour le web.
